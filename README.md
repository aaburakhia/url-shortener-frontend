# Serverless URL Shortener on AWS with Streamlit Frontend

This is a complete, full-stack URL shortener application built to demonstrate a modern, scalable, and cost-effective serverless architecture on Amazon Web Services (AWS). It features a powerful serverless backend and a user-friendly web interface built with Streamlit.

The entire AWS backend is designed to run completely within the AWS Free Tier for typical use cases.

**Live Application:** [You can view the live Streamlit frontend here!](https://url-shortener-testapp.streamlit.app)
